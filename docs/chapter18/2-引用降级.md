## 降级是安全的

把一个"活得更长"的引用用在"需要活得短一点"的位置，是安全的：

```rust
fn short_term(s: &str) { println!("{s}"); }

let owned = String::from("hello");
{
    let view: &str = &owned;  // view 的生命周期在这个块内
    short_term(view);          // ✅ 传给需要任意 'a 的函数

    // owned 活得比 view 长，没问题
}

// 反过来不行：
// fn long_term(s: &'static str) { ... }
// long_term(&owned); // ❌ &owned 不是 'static
```

---

## 直觉

> "更长寿的引用可以临时被当作更短寿的使用"——就像你 100 岁的老人可以做 30 岁的工作，但反过来 30 岁的人无法做 100 岁才有资格做的事情。

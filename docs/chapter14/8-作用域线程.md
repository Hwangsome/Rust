# 8. 作用域线程 `thread::scope`

## 一分钟结论

`thread::scope(|s| { s.spawn(...); })` 保证所有子线程在 scope 结束前被 join，所以**子线程可以安全借用父栈上的变量**，不需要 Arc / clone。

## 对应代码

- [topic_08_scoped_threads.rs](../../chapters/chapter14/src/topic_08_scoped_threads.rs)

## 对比

- 普通 `thread::spawn` → `'static` 约束强，需要 Arc / clone / move
- `thread::scope` → 可借用栈数据，代码更干净

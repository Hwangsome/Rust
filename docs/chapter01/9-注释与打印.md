## 这一节解决什么问题

Rust 的注释和打印比其他语言多了几个重要概念：

- 注释除了给人看，还有**文档注释**（`///` 和 `//!`）可以生成 HTML 文档
- `println!` 是**宏**（不是函数），支持丰富的格式化语法
- `{:?}`（Debug）和 `{}`（Display）是两种不同的格式化方式——理解区别很重要

---

## 一分钟结论

**注释类型**：
- `//` 行注释（最常用）
- `/* ... */` 块注释
- `///` 文档注释（写在 pub item 前，生成 API 文档）
- `//!` 内层文档注释（写在文件/模块顶部，描述自身）

**打印宏**：
- `print!` / `println!`：输出到标准输出（stdout）
- `eprint!` / `eprintln!`：输出到标准错误（stderr）
- `format!`：格式化成 String，不打印
- `{}`：Display 格式（给用户看）
- `{:?}`：Debug 格式（给开发者看）
- `{:#?}`：Pretty Debug（自动换行缩进）

---

## 与其他语言对比

| 特性 | Java | Python | Rust |
|-----|------|--------|------|
| 文档注释 | `/** ... */`（Javadoc）| `"""..."""`（docstring）| `///`（rustdoc）|
| 格式化字符串 | `String.format()` | f-strings / `%` / `.format()` | `format!()` |
| 默认打印 | `System.out.println()` | `print()` | `println!()` |
| 调试打印 | `toString()` 或 IDE | `repr()` / `str()` | `{:?}`（需要 `#[derive(Debug)]`）|

## 详细原理

### 1. 注释类型

```rust
// 这是行注释——最常用

/* 这是块注释
   可以跨多行
   但不能嵌套另一个 /* 块注释 */
*/

/// 这是文档注释，写在 pub 函数、struct、模块前面
/// 支持 Markdown 格式
/// 
/// # Examples
/// 
/// ```
/// let result = my_function(5);
/// assert_eq!(result, 10);
/// ```
pub fn my_function(n: i32) -> i32 { n * 2 }

//! 这是内层文档注释，通常写在文件顶部
//! 描述这个模块/文件本身的功能
```

`cargo doc --open` 会把 `///` 和 `//!` 注释生成漂亮的 HTML 文档。

### 2. 格式化占位符

```rust
let name = "Alice";
let age = 30;
let height = 1.75;

// {} → Display（要求类型实现 Display trait）
println!("{name} 今年 {age} 岁");

// {:?} → Debug（要求类型实现 Debug trait）
let v = vec![1, 2, 3];
println!("{v:?}");    // [1, 2, 3]

// {:#?} → Pretty Debug（自动缩进）
println!("{v:#?}");
// [
//     1,
//     2,
//     3,
// ]

// 格式化数字
println!("{:.2}", 3.14159);    // 3.14（保留2位小数）
println!("{:>10}", "right");   // "     right"（右对齐，宽度10）
println!("{:<10}", "left");    // "left      "（左对齐）
println!("{:^10}", "center");  // "  center  "（居中）
println!("{:0>5}", 42);        // "00042"（补零）
println!("{:b}", 255);         // "11111111"（二进制）
println!("{:x}", 255);         // "ff"（十六进制小写）
println!("{:X}", 255);         // "FF"（十六进制大写）
println!("{:e}", 1234567.0);   // "1.234567e6"（科学计数法）
```

### 3. `Display` vs `Debug`

```rust
use std::fmt;

struct Point { x: f64, y: f64 }

// Display：给用户看的格式
impl fmt::Display for Point {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "({}, {})", self.x, self.y)
    }
}

// Debug：派生宏自动实现
#[derive(Debug)]
struct PointD { x: f64, y: f64 }

let p = Point { x: 1.0, y: 2.0 };
let pd = PointD { x: 1.0, y: 2.0 };

println!("{p}");    // (1, 2)        ← Display：美观
println!("{pd:?}"); // PointD { x: 1.0, y: 2.0 } ← Debug：包含类型信息
```

## 完整运行示例

```rust
fn main() {
    // === 1. 注释 ===
    // 行注释：单行说明
    let x = 5; // 行尾注释

    /* 块注释：适合临时禁用代码
    let y = 10;
    let z = y + x;
    */

    println!("=== 基础打印 ===");
    println!("hello, world");    // 换行
    print!("no newline");        // 不换行
    print!(" here\n");           // 手动换行
    eprintln!("这条去 stderr");  // stderr（常用于错误/日志）

    println!();

    println!("=== 格式化占位符 ===");
    let pi = std::f64::consts::PI;
    println!("Pi = {pi}");             // 默认精度
    println!("Pi = {pi:.2}");          // 2位小数
    println!("Pi = {pi:.5}");          // 5位小数
    println!("Pi = {:10.3}", pi);      // 宽度10，3位小数

    println!();

    println!("=== 对齐和填充 ===");
    println!("|{:>10}|", "right");
    println!("|{:<10}|", "left");
    println!("|{:^10}|", "center");
    println!("|{:*>10}|", "fill");     // 用 * 填充
    println!("|{:0>5}|", 42);          // 补零

    println!();

    println!("=== 进制 ===");
    let n = 255_u8;
    println!("十进制: {n}");
    println!("二进制: {:b}", n);
    println!("八进制: {:o}", n);
    println!("十六进制: {:x} / {:X}", n, n);

    println!();

    println!("=== Debug vs Display ===");
    let nums = vec![1, 2, 3];
    let text = "hello";

    println!("Display: {} {:?}", text, nums);   // Debug 需要 {:?}
    // println!("{}", nums); // ❌ Vec 没有实现 Display

    println!("Pretty debug:");
    println!("{:#?}", nums);

    println!();

    println!("=== format! 构建字符串 ===");
    let greeting = format!("Hello, {}! You are {} years old.", "Alice", 30);
    println!("{greeting}");

    // 位置参数
    println!("{0}, {1}, {0}", "tic", "tac");  // tic, tac, tic

    // 命名参数
    println!("{name} has {count} items", name = "Bob", count = 5);
}
```

## 注意点与陷阱

### 陷阱 1：`println!` 格式字符串里的 `{` 需要用 `{{` 转义

```rust
println!("{{}});     // 打印: {}
println!("{{{}}}",  42); // 打印: {42}
```

### 陷阱 2：`Debug` 和 `Display` 的格式可能随版本变化

```rust
// 不要依赖 {:?} 输出的具体格式来解析
// 它只保证"有意义的调试信息"，不保证格式稳定
```

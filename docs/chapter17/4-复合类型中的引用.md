## struct 字段是引用时的生命周期

```rust
// 引用字段必须有生命周期参数
struct TextSlice<'a> {
    content: &'a str,
    position: usize,
}

impl<'a> TextSlice<'a> {
    fn new(text: &'a str, pos: usize) -> Self {
        TextSlice { content: text, position: pos }
    }
    fn preview(&self) -> &str {
        &self.content[..self.content.len().min(20)]
    }
}

let text = String::from("Hello, Rust! How are you?");
let slice = TextSlice::new(&text, 0);
println!("{}", slice.preview());
```

## Vec 和 HashMap 里的引用

```rust
// Vec<&str>：元素是引用，生命周期受约束
let s1 = String::from("alice");
let s2 = String::from("bob");
let names: Vec<&str> = vec![s1.as_str(), s2.as_str()];
println!("{names:?}");
// names 不能活过 s1 / s2

// HashMap<&str, usize>
let words = vec!["rust", "is", "fast", "rust"];
let mut freq = std::collections::HashMap::new();
for w in &words {
    *freq.entry(w).or_insert(0) += 1;
}
println!("{freq:?}");
```

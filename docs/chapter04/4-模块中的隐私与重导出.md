## 这一节解决什么问题

`pub struct Foo { ... }` 只公开了**类型名**，字段仍然私有。这让你能控制"如何构造这个类型"——通过提供 `new()` 等构造函数来加校验。

同时，`pub use` 让你"重塑" API 的对外形状——内部可以复杂，对外暴露的路径可以简洁。

---

## 一分钟结论

- `pub struct` 公开类型名，字段仍需单独 `pub` 才能外部访问
- 字段私有 + 公开构造函数 = **封装**（保证不变量）
- `pub enum` 的变体**默认**跟着 `pub`（与 struct 字段不同）
- `pub use mod::Type` = **重导出**：让 `Type` 在当前模块路径可访问

---

## 详细原理

### 1. 字段私有 + 构造函数控制

```rust
mod user_module {
    #[derive(Debug)]
    pub struct User {
        username: String,     // 私有
        email: String,        // 私有
        pub is_admin: bool,   // 公开（特意允许外部读）
    }

    impl User {
        // 构造函数做校验
        pub fn new(username: &str, email: &str) -> Result<Self, String> {
            if !email.contains('@') {
                return Err(format!("无效邮箱: {email}"));
            }
            Ok(User {
                username: username.to_string(),
                email: email.to_string(),
                is_admin: false,
            })
        }

        // getter（受控读取私有字段）
        pub fn username(&self) -> &str { &self.username }
        pub fn email(&self) -> &str { &self.email }
    }
}

// 外部使用：
use user_module::User;
// let u = User { username: "alice".into(), ... }; // ❌ 字段私有
let u = User::new("alice", "alice@example.com").unwrap(); // ✅ 通过构造函数
println!("{}", u.username());  // ✅ 通过 getter
println!("{}", u.is_admin);    // ✅ is_admin 是 pub
```

### 2. `pub use` 重导出

```rust
// 内部结构复杂：
mod internal {
    pub mod types {
        pub struct Config { pub host: String }
    }
    pub mod utils {
        pub fn parse(s: &str) -> i32 { s.parse().unwrap_or(0) }
    }
}

// 用 pub use 整理对外 API：
pub use internal::types::Config;   // 暴露为 crate::Config
pub use internal::utils::parse;    // 暴露为 crate::parse

// 外部用户直接用：
// use my_crate::Config;  （不需要知道内部路径）
```

## 完整运行示例

```rust
mod products {
    #[derive(Debug, Clone)]
    pub struct Product {
        id: u32,
        name: String,
        price_cents: u32,  // 用分避免浮点精度问题
        in_stock: bool,
    }

    impl Product {
        pub fn new(id: u32, name: &str, price_cents: u32) -> Self {
            Product {
                id, name: name.into(), price_cents, in_stock: true
            }
        }

        pub fn id(&self) -> u32 { self.id }
        pub fn name(&self) -> &str { &self.name }
        pub fn price_display(&self) -> String {
            format!("${:.2}", self.price_cents as f64 / 100.0)
        }
        pub fn in_stock(&self) -> bool { self.in_stock }

        pub fn apply_discount(&mut self, percent: u32) {
            self.price_cents = self.price_cents * (100 - percent) / 100;
        }
    }
}

// 重导出，简化用户路径
pub use products::Product;

pub fn run() {
    println!("=== pub struct：字段私有 ===");
    let mut p = Product::new(1, "Rust 书籍", 3999);

    // ✅ 通过公开方法访问
    println!("{} - {} (库存: {})", p.name(), p.price_display(), p.in_stock());

    // 应用折扣
    p.apply_discount(10);  // 九折
    println!("折后: {}", p.price_display());
    println!();

    println!("=== pub enum：变体默认公开 ===");
    #[derive(Debug)]
    pub enum Status { Active, Inactive, Suspended(String) }

    let s = Status::Suspended("违规".into());
    match s {
        Status::Active => println!("活跃"),
        Status::Inactive => println!("不活跃"),
        Status::Suspended(reason) => println!("暂停: {reason}"),
    }
}
```

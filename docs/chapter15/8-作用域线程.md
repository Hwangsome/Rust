## 普通线程的限制

`thread::spawn` 要求 `F: 'static`——不能借用栈上局部变量（因为线程可能活过当前栈帧）。

---

## `thread::scope`：安全借用局部变量

```rust
use std::thread;

let numbers = vec![1, 2, 3, 4, 5];
let prefix = "数字";

thread::scope(|s| {
    // 这些线程可以借用 numbers 和 prefix！
    s.spawn(|| {
        let sum: i32 = numbers.iter().sum();
        println!("{prefix}之和: {sum}");
    });

    s.spawn(|| {
        let max = numbers.iter().max().unwrap();
        println!("{prefix}最大值: {max}");
    });
    // scope 结束前，所有子线程都会被自动 join
});

// scope 结束后，numbers 和 prefix 仍然可用
println!("numbers: {numbers:?}");
```

---

## 为什么安全

`thread::scope` 保证：在 scope 闭包返回之前，所有 scoped 线程都会被 join。因此，子线程借用的数据一定活得足够长。

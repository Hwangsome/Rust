# 1. 引用的种类速查

| 种类 | 特征 | 用途 |
|-----|------|-----|
| `&T` | 只读借用，`Copy` | 最常见的读接口 |
| `&mut T` | 独占可变借用，非 Copy | 修改接口 |
| `Box<T>` | 堆上独占 | 大对象 / trait object / 递归类型 |
| `Rc<T>` | 单线程共享所有权 | 树 / 图 / 缓存 |
| `Arc<T>` | 多线程共享所有权 | 跨线程数据 |
| `*const T` / `*mut T` | 裸指针，不受 borrow checker 管 | FFI / unsafe |

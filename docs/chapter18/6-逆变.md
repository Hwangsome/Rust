## 什么是逆变

`F<T>` 在 T 上逆变（contravariant）：如果 `A: B`，则 `F<B>: F<A>`（关系反转）。

---

## 函数参数是逆变的

```rust
// fn(T) 在 T 上逆变：
// - fn(&'any str)：能接受任何寿命的 &str
// - fn(&'static str)：只能接受 'static 的 &str

// 能"接受任何"的函数更灵活，是"只接受特定"函数的子类型：
let broad: fn(&str) = |s| println!("{s}");
let narrow: fn(&'static str) = broad; // ✅ broad 更灵活，可以用于 narrow 的位置
// let rev: fn(&str) = narrow; // ❌ 反过来不行
```

---

## 直觉

> "能做更多的工具可以替代只能做部分的工具，反过来不行。"
> 能处理任意字符串的函数，当然可以胜任只处理静态字符串的工作。

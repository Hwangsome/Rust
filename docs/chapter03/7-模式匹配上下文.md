## 这一节解决什么问题

"模式匹配"不只是 `match`——它出现在 Rust 的很多地方：`let`、函数参数、`for` 循环、`if let`、`while let`。

理解"模式"这个概念，能让你写出更简洁、更安全的代码。

---

## 一分钟结论

- `match` 是最全面的模式匹配，**穷尽检查**（编译器保证）
- `if let`：只关心一种情况（单分支 match 的语法糖）
- `while let`：循环直到模式不匹配
- `let` 解构：`let (a, b) = (1, 2);`，`let [x, y, ..] = arr;`
- 守卫（guard）：`n if n > 0 => ...`
- `@` 绑定：`n @ 1..=9 => ...`（绑定值同时匹配范围）
- `|` 或模式：`1 | 2 | 3 => ...`

## 详细原理

### 1. `match` 进阶

```rust
let n = 7;
match n {
    // 字面量
    0 => println!("零"),

    // 范围
    1..=5 => println!("1到5"),

    // 或模式
    6 | 7 | 8 => println!("6、7或8"),

    // 守卫（guard）
    x if x % 2 == 0 => println!("{x} 是偶数"),

    // @ 绑定：同时匹配范围并绑定变量
    n @ 10..=19 => println!("十几: {n}"),

    // 通配符
    _ => println!("其他"),
}
```

### 2. `if let`

```rust
let config: Option<String> = Some("debug".to_string());

// match 的冗长写法
match config {
    Some(c) if c == "debug" => println!("调试模式: {c}"),
    Some(c) => println!("模式: {c}"),
    None => {}  // 什么都不做，但必须写
}

// if let 简洁写法（只关心 Some 的情况）
if let Some(mode) = &config {
    println!("配置: {mode}");
}

// if let ... else
if let Some(mode) = &config {
    println!("有配置: {mode}");
} else {
    println!("无配置，使用默认");
}
```

### 3. `while let`

```rust
let mut stack = vec![1, 2, 3, 4, 5];

while let Some(top) = stack.pop() {
    println!("弹出: {top}");
    // 直到 stack 为空，pop() 返回 None，循环结束
}
println!("stack 清空了");
```

### 4. `let` 解构

```rust
// 元组解构
let (x, y, z) = (1, 2.0, "three");
println!("{x} {y} {z}");

// 忽略某些字段
let (first, _, third) = (1, 2, 3);

// 数组解构（含 rest 模式）
let [head, tail @ ..] = [1, 2, 3, 4, 5];
println!("head={head}, tail={tail:?}");

// struct 解构
struct Point { x: i32, y: i32 }
let p = Point { x: 10, y: 20 };
let Point { x, y } = p;
println!("x={x}, y={y}");
```

## 完整运行示例

```rust
#[derive(Debug)]
enum Event {
    MouseClick { x: i32, y: i32 },
    KeyPress(char),
    Scroll { delta: f64, direction: Direction },
    Quit,
}

#[derive(Debug)]
enum Direction { Up, Down }

fn handle_event(event: &Event) -> String {
    match event {
        // struct 变体解构
        Event::MouseClick { x, y } if *x > 0 && *y > 0 => {
            format!("点击第一象限: ({x}, {y})")
        }
        Event::MouseClick { x, y } => {
            format!("点击: ({x}, {y})")
        }

        // 元组变体解构
        Event::KeyPress(c @ 'a'..='z') => format!("小写字母: {c}"),
        Event::KeyPress(c @ 'A'..='Z') => format!("大写字母: {c}"),
        Event::KeyPress(c) => format!("按键: {c}"),

        // 嵌套解构
        Event::Scroll { direction: Direction::Up, delta } => {
            format!("上滚: {delta:.1}")
        }
        Event::Scroll { direction: Direction::Down, delta } => {
            format!("下滚: {delta:.1}")
        }

        Event::Quit => "退出".to_string(),
    }
}

fn main() {
    let events = vec![
        Event::MouseClick { x: 100, y: 200 },
        Event::MouseClick { x: -10, y: 50 },
        Event::KeyPress('a'),
        Event::KeyPress('Z'),
        Event::KeyPress('5'),
        Event::Scroll { delta: 3.5, direction: Direction::Up },
        Event::Quit,
    ];

    println!("=== 事件处理 ===");
    for event in &events {
        println!("  {:?} → {}", event, handle_event(event));
    }
    println!();

    println!("=== while let ===");
    let mut events2 = events.into_iter();
    while let Some(e) = events2.next() {
        if let Event::Quit = e { println!("  收到 Quit，停止"); break; }
        print!("  处理中... ");
    }
    println!();

    println!("=== let 解构 ===");
    let points = [(1, 2), (3, 4), (5, 6)];
    for (x, y) in &points {
        println!("  ({x}, {y})");
    }
}
```

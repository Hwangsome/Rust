## 这一节解决什么问题

如何写 Rust 测试？测试写在哪里？和被测代码放在一起还是分离？

Rust 的单元测试特别之处：**和被测代码放在同一个文件里**，用 `#[cfg(test)]` 标记测试模块——只在 `cargo test` 时编译，不会进入生产二进制。

---

## 一分钟结论

- `#[test]` 标记测试函数
- `#[cfg(test)] mod tests { ... }` 标记测试模块（仅在测试时编译）
- `assert!(condition)` / `assert_eq!(a, b)` / `assert_ne!(a, b)` 断言
- `#[should_panic]` / `#[should_panic(expected = "...")]` 期望 panic
- 测试函数可以返回 `Result<(), E>`，允许用 `?` 传播错误
- `cargo test` 运行所有测试；`cargo test func_name` 运行特定测试

## 详细原理

### 1. 基础测试

```rust
fn add(a: i32, b: i32) -> i32 { a + b }
fn is_even(n: i32) -> bool { n % 2 == 0 }

#[cfg(test)]
mod tests {
    use super::*;  // 导入被测代码

    #[test]
    fn test_add() {
        assert_eq!(add(2, 3), 5);
        assert_eq!(add(-1, 1), 0);
        assert_ne!(add(2, 2), 5);  // 不等断言
    }

    #[test]
    fn test_is_even() {
        assert!(is_even(4));
        assert!(!is_even(3));
    }
}
```

### 2. 断言消息

```rust
#[test]
fn test_with_message() {
    let x = 5;
    assert!(x > 0, "x 应该大于 0，实际值: {x}");
    assert_eq!(x, 5, "期望 x = 5，实际 x = {x}");
}
```

### 3. `should_panic`

```rust
fn divide(a: i32, b: i32) -> i32 {
    if b == 0 { panic!("除以零！"); }
    a / b
}

#[test]
#[should_panic(expected = "除以零")]  // 验证 panic 消息包含这段文字
fn test_divide_by_zero() {
    divide(10, 0);
}
```

### 4. 返回 `Result` 的测试

```rust
#[test]
fn test_parse() -> Result<(), std::num::ParseIntError> {
    let n: i32 = "42".parse()?;  // 用 ? 传播错误
    assert_eq!(n, 42);
    Ok(())
}
```

## 完整运行示例

```rust
pub fn add(a: i32, b: i32) -> i32 { a + b }
pub fn divide(a: f64, b: f64) -> Option<f64> {
    if b == 0.0 { None } else { Some(a / b) }
}

#[cfg(test)]
mod tests {
    use super::*;

    // 基础断言
    #[test]
    fn test_add_positive() { assert_eq!(add(2, 3), 5); }

    #[test]
    fn test_add_negative() { assert_eq!(add(-1, -2), -3); }

    // Option 测试
    #[test]
    fn test_divide_ok() {
        assert_eq!(divide(10.0, 2.0), Some(5.0));
    }

    #[test]
    fn test_divide_by_zero() {
        assert_eq!(divide(10.0, 0.0), None);
    }

    // Result 风格
    #[test]
    fn test_with_result() -> Result<(), String> {
        let result = divide(10.0, 2.0).ok_or("除以零")?;
        assert!((result - 5.0).abs() < f64::EPSILON);
        Ok(())
    }

    // 帮助函数
    fn setup() -> Vec<i32> { vec![1, 2, 3, 4, 5] }

    #[test]
    fn test_with_setup() {
        let data = setup();
        assert_eq!(data.len(), 5);
        assert_eq!(data.iter().sum::<i32>(), 15);
    }
}
```

## 常用 assert 宏

| 宏 | 用途 |
|---|-----|
| `assert!(expr)` | expr 为 true |
| `assert_eq!(a, b)` | a == b |
| `assert_ne!(a, b)` | a != b |
| `assert!(expr, "msg")` | expr 为 true，失败时显示 msg |
| `assert_eq!(a, b, "msg")` | a == b，失败时显示 msg |

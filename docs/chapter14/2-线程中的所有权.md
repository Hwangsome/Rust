# 2. 线程中的所有权：`move`、`Send`、`Sync`

## 一分钟结论

- `thread::spawn(F)` 要求 `F: Send + 'static`
- `'static`：闭包不能借用比 static 还短寿的外部引用
- `Send`：闭包 + 捕获的所有东西都能跨线程
- 捕获外部变量一般要写 `move`——把所有权搬进闭包

## 速记

- `Rc<T>` 不是 Send → 多线程必须用 `Arc<T>`
- `RefCell<T>` 不是 Sync → 多线程必须用 `Mutex<T>` / `RwLock<T>`

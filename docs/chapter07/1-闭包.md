# 1. 闭包

- Cargo package: `chapter07`
- Run chapter: `cargo run -p chapter07`

- Reference module: `chapters/chapter07/src/closures.rs`
- Chapter lab: `chapters/chapter07/src/lab.rs`

## 定义

闭包是匿名的可调用值。它看起来像函数，但还能捕获定义处周围的环境。

## 作用

- 把短小行为当成参数传入函数
- 延迟决定具体逻辑
- 在不额外定义命名函数的情况下携带上下文

## 原理

闭包会根据用法自动推断参数和返回值，并根据是否捕获环境、是否修改环境，分别实现 `Fn`、`FnMut`、`FnOnce` 等 trait。

## 最小示例

```rust
let min_length = 4;
let is_valid_user = |name: &str| name.len() >= min_length;
```

## 注意点

- 闭包和函数最大的区别是“能否捕获环境”
- 捕获方式会影响它实现的是哪一个 `Fn` trait
- 零捕获闭包有时可以退化成函数指针

## 常见错误

- 把闭包当成普通函数理解，忽略外部变量捕获
- 不看签名约束，就把任意闭包传给函数
- 遇到 move 闭包时不知道为什么外部值被拿走

## 我的理解

闭包让“行为”真正成为一等值。后面很多迭代器组合子之所以顺手，本质上就是因为闭包能很自然地嵌进去。

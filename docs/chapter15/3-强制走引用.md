# 3. 强制 for 循环走借用

| 写法 | 产出 | 是否消费 |
|------|-----|--------|
| `for x in v` | `T` | ✅ |
| `for x in &v` | `&T` | ❌ |
| `for x in &mut v` | `&mut T` | ❌ |
| `v.iter()` / `iter_mut()` / `into_iter()` | 对应类型 | 同上 |

想保留集合 → 加 `&`；想改元素 → `&mut`；明确丢掉 → `into_iter`。

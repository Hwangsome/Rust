## 使用 `dyn Trait` 时必须指定关联类型

```rust
use std::fmt::Debug;

// ❌ 不行：Iterator 有关联类型 Item，必须指定
// let iter: Box<dyn Iterator> = Box::new(1..=5);

// ✅ 必须指定 Item
let iter: Box<dyn Iterator<Item = i32>> = Box::new(1..=5);
let sum: i32 = iter.sum();
println!("sum = {sum}");
```

---

## 原因

编译器需要知道关联类型才能生成 vtable。如果不指定 `Item`，`next()` 的返回类型就不确定，无法生成函数指针。

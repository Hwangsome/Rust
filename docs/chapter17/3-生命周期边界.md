## 语法

`'a: 'b` 读作 "`'a` outlives `'b`"，意思是 `'a` 至少活得和 `'b` 一样长。

```rust
// 这个函数：
// - 接受两个引用（可能不同生命周期）
// - 返回引用，其生命周期不超过两个输入中较短的那个
fn longest<'a, 'b: 'a>(x: &'a str, y: &'b str) -> &'a str {
    if x.len() >= y.len() { x } else { y }
}
```

---

## 结构体里的 outlives

```rust
struct Important<'a> {
    content: &'a str,
}

// 这个 impl 里，T 必须活得不短于 'a
impl<'a, T: 'a> Important<'a> {
    fn announce(&self, extra: &T) where T: std::fmt::Debug {
        println!("{}: {:?}", self.content, extra);
    }
}
```

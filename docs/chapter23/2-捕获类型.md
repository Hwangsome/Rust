## 捕获类型速查

| 指示符 | 匹配 | 示例 |
|--------|-----|-----|
| `expr` | 表达式 | `1 + 2`、`"str"` |
| `ident` | 标识符 | `foo`、`my_var` |
| `ty` | 类型 | `i32`、`Vec<u8>` |
| `literal` | 字面量 | `42`、`"hello"` |
| `pat` | 模式 | `Some(x)`、`1..=5` |
| `path` | 路径 | `std::io::Error` |
| `stmt` | 语句 | `let x = 5;` |
| `block` | 块 | `{ foo(); bar() }` |
| `item` | 项目 | `fn foo() {}` |
| `tt` | token tree（最通用）| 任意 |

## 使用示例

```rust
macro_rules! log_value {
    ($name:ident = $value:expr) => {
        let $name = $value;
        println!("[LOG] {} = {:?}", stringify!($name), $name);
    };
}

log_value!(answer = 6 * 7);
// 展开为：
// let answer = 6 * 7;
// println!("[LOG] {} = {:?}", "answer", answer);

macro_rules! make_fn {
    ($name:ident, $ret_ty:ty, $body:block) => {
        fn $name() -> $ret_ty $body
    };
}

make_fn!(get_pi, f64, { 3.14159 });
println!("{}", get_pi()); // 3.14159
```

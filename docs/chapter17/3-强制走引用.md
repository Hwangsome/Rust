## 三种 for 循环语义

```rust
let v = vec![String::from("a"), String::from("b")];

// for x in v         → 消耗 v，x 是 String（拥有）
// for x in &v        → 借用 v，x 是 &String（只读）
// for x in &mut v    → 可变借用 v，x 是 &mut String（可改）
```

---

## 详细对比

```rust
let strings = vec![String::from("hello"), String::from("world")];

// 消耗（不保留原集合）
for s in strings {
    println!("{s}"); // s: String（有所有权）
}
// println!("{strings:?}"); // ❌ strings 已被消耗

// 只读借用（保留原集合）
let strings = vec![String::from("hi"), String::from("there")];
for s in &strings {
    println!("{s}"); // s: &String
}
println!("{strings:?}"); // ✅ 仍然有效

// 可变借用（原地修改）
let mut nums = vec![1, 2, 3];
for n in &mut nums {
    *n *= 2; // n: &mut i32
}
println!("{nums:?}"); // [2, 4, 6]，仍然有效
```

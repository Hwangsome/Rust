## 移除头节点时需要更新 prev

单向链表的头删只需处理 head 指针。双向链表还需要：
1. 断开旧 head 的 next 的 prev 引用（否则形成悬垂引用）

```rust
fn pop_front(&mut self) -> Option<i32> {
    self.head.take().map(|old_head| {
        // old_head 现在是唯一的 Rc 引用（size 降到 1）
        let value = old_head.borrow().value;
        let next_node = old_head.borrow_mut().next.take(); // 切断 old_head → next 的连接

        if let Some(next) = &next_node {
            next.borrow_mut().prev = None; // 断开 next → old_head 的 prev 连接
        } else {
            self.tail = None; // 链表现在为空
        }

        self.head = next_node;
        self.size -= 1;
        value
    })
}
```

## 遍历

```rust
fn values(&self) -> Vec<i32> {
    let mut result = Vec::new();
    let mut current = self.head.clone();

    while let Some(node_rc) = current {
        result.push(node_rc.borrow().value);
        current = node_rc.borrow().next.clone(); // clone Rc（不 clone 数据）
    }

    result
}
```

## `park()` / `unpark()`：低级同步原语

```rust
use std::thread;
use std::time::Duration;

let worker = thread::spawn(|| {
    println!("  工作线程: 开始工作");
    thread::park();  // 暂停，等待唤醒
    println!("  工作线程: 被唤醒，继续工作");
});

thread::sleep(Duration::from_millis(50));
println!("  主线程: 唤醒工作线程");
worker.thread().unpark(); // 唤醒

worker.join().unwrap();
```

---

## 注意

- `park()` 可能**假唤醒**（spurious wake-up）—— 总是在循环里检查条件
- 实际工程中优先用 `channel` 或 `Mutex` + `Condvar`
- `park/unpark` 适合实现自定义同步工具

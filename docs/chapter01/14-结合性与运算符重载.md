## 这一节解决什么问题

两个关联问题：

1. **结合性**：`20 - 5 - 3` 等于 `(20 - 5) - 3 = 12`，还是 `20 - (5 - 3) = 18`？
2. **运算符重载**：为什么可以写 `let sum = point1 + point2`？

---

## 一分钟结论

**结合性**：
- 大多数二元运算符是**左结合**：`a - b - c` 等于 `(a - b) - c`
- 赋值是**右结合**（但 Rust 的赋值不返回值，所以链式赋值不常见）
- 一元运算符（如 `-x`）是右结合

**运算符重载**：
- 通过实现 `std::ops` 里的 trait 来重载运算符
- `+` → `impl Add`；`-` → `impl Sub`；`*` → `impl Mul`；等等
- 重载后使用方式和内置类型完全一样

---

## 详细原理

### 结合性

```rust
// 左结合（大多数运算符）
let a = 20 - 5 - 3;    // (20-5)-3 = 12（从左到右）
let b = 20 - (5 - 3);  // 20-2 = 18（括号改变分组）
println!("20-5-3 = {a}");     // 12
println!("20-(5-3) = {b}");   // 18

// 乘法也是左结合
let c = 2 * 3 * 4;  // (2*3)*4 = 24

// 幂运算在很多语言是右结合，但 Rust 没有内置幂运算符
// 用 pow() 函数替代：
let d = 2_i32.pow(3).pow(2); // 8^2 = 64（但注意：Rust 里这是函数调用，不是运算符）
```

### 运算符重载

```rust
use std::ops::{Add, Mul, Neg, Sub};
use std::fmt;

#[derive(Debug, Clone, Copy, PartialEq)]
struct Vec2 {
    x: f64,
    y: f64,
}

impl Vec2 {
    fn new(x: f64, y: f64) -> Self { Vec2 { x, y } }
    fn magnitude(&self) -> f64 { (self.x.powi(2) + self.y.powi(2)).sqrt() }
}

impl fmt::Display for Vec2 {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "({:.1}, {:.1})", self.x, self.y)
    }
}

impl Add for Vec2 {
    type Output = Vec2;
    fn add(self, rhs: Vec2) -> Vec2 {
        Vec2::new(self.x + rhs.x, self.y + rhs.y)
    }
}

impl Sub for Vec2 {
    type Output = Vec2;
    fn sub(self, rhs: Vec2) -> Vec2 {
        Vec2::new(self.x - rhs.x, self.y - rhs.y)
    }
}

impl Mul<f64> for Vec2 {
    type Output = Vec2;
    fn mul(self, scalar: f64) -> Vec2 {
        Vec2::new(self.x * scalar, self.y * scalar)
    }
}

impl Neg for Vec2 {
    type Output = Vec2;
    fn neg(self) -> Vec2 {
        Vec2::new(-self.x, -self.y)
    }
}
```

## 完整运行示例

```rust
use std::ops::{Add, Mul, Neg, Sub};
use std::fmt;

#[derive(Debug, Clone, Copy, PartialEq)]
struct Vec2 { x: f64, y: f64 }

impl Vec2 {
    fn new(x: f64, y: f64) -> Self { Vec2 { x, y } }
    fn magnitude(&self) -> f64 { (self.x * self.x + self.y * self.y).sqrt() }
    fn dot(&self, other: &Vec2) -> f64 { self.x * other.x + self.y * other.y }
}

impl fmt::Display for Vec2 {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        write!(f, "({:.1}, {:.1})", self.x, self.y)
    }
}

impl Add for Vec2 {
    type Output = Vec2;
    fn add(self, rhs: Vec2) -> Vec2 { Vec2::new(self.x + rhs.x, self.y + rhs.y) }
}

impl Sub for Vec2 {
    type Output = Vec2;
    fn sub(self, rhs: Vec2) -> Vec2 { Vec2::new(self.x - rhs.x, self.y - rhs.y) }
}

impl Mul<f64> for Vec2 {
    type Output = Vec2;
    fn mul(self, s: f64) -> Vec2 { Vec2::new(self.x * s, self.y * s) }
}

impl Neg for Vec2 {
    type Output = Vec2;
    fn neg(self) -> Vec2 { Vec2::new(-self.x, -self.y) }
}

fn main() {
    println!("=== 结合性 ===");
    let a = 20 - 5 - 3;
    let b = 20 - (5 - 3);
    println!("20-5-3     = {a}  (左结合: (20-5)-3)");
    println!("20-(5-3)   = {b}  (括号改变了分组)");
    println!();

    println!("=== 运算符重载 ===");
    let v1 = Vec2::new(3.0, 4.0);
    let v2 = Vec2::new(1.0, 2.0);

    println!("v1 = {v1}, 模长 = {:.2}", v1.magnitude());
    println!("v2 = {v2}");
    println!("v1 + v2 = {}", v1 + v2);
    println!("v1 - v2 = {}", v1 - v2);
    println!("v1 * 2.0 = {}", v1 * 2.0);
    println!("-v1 = {}", -v1);
    println!("v1 · v2 = {:.1} (点积)", v1.dot(&v2));
    println!();

    // 可以链式调用
    let result = (v1 + v2) * 0.5;
    println!("(v1 + v2) * 0.5 = {result} (中点)");
}
```

## 常用 `std::ops` Trait

| 运算符 | Trait | 实现方法 |
|-------|-------|---------|
| `a + b` | `Add<Rhs>` | `fn add(self, rhs: Rhs) -> Output` |
| `a - b` | `Sub<Rhs>` | `fn sub(self, rhs: Rhs) -> Output` |
| `a * b` | `Mul<Rhs>` | `fn mul(self, rhs: Rhs) -> Output` |
| `a / b` | `Div<Rhs>` | `fn div(self, rhs: Rhs) -> Output` |
| `a % b` | `Rem<Rhs>` | `fn rem(self, rhs: Rhs) -> Output` |
| `-a` | `Neg` | `fn neg(self) -> Output` |
| `!a` | `Not` | `fn not(self) -> Output` |
| `a[i]` | `Index` | `fn index(&self, idx: Idx) -> &Output` |
| `a[i] = v` | `IndexMut` | `fn index_mut(&mut self, idx: Idx) -> &mut Output` |
| `a += b` | `AddAssign` | `fn add_assign(&mut self, rhs: Rhs)` |

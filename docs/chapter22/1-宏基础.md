## 什么是声明式宏

`macro_rules!` 定义的宏是基于**模式匹配的代码生成**：

```rust
macro_rules! greet {
    () => {
        println!("Hello!");
    };
    ($name:expr) => {
        println!("Hello, {}!", $name);
    };
}

greet!();           // Hello!
greet!("Alice");    // Hello, Alice!
```

---

## 宏与函数的区别

| 特性 | 函数 | macro_rules! |
|-----|-----|-------------|
| 参数数量 | 固定 | 可变（重复模式）|
| 参数类型 | 确定 | 任意 Rust 语法片段 |
| 展开时机 | 运行时 | **编译时** |
| 能生成代码 | 否 | ✅（比如 `vec!` 生成 Vec 初始化）|
| 卫生性 | N/A | 宏卫生（hygienic）|

---

## 完整示例

```rust
macro_rules! create_function {
    ($fn_name:ident) => {
        fn $fn_name() {
            println!("函数 {:?} 被调用", stringify!($fn_name));
        }
    };
}

// 生成 foo() 和 bar() 两个函数
create_function!(foo);
create_function!(bar);

foo(); // 函数 "foo" 被调用
bar(); // 函数 "bar" 被调用
```

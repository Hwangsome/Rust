## 什么是生命周期子类型

在类型理论里，"A 是 B 的子类型"意味着"A 可以用在任何需要 B 的地方"。

对生命周期：`'long: 'short`（'long outlives 'short）意味着 `'long` **至少活得和 `'short` 一样长**。

```
'static 是最长的生命周期
'static: 'a（对任意 'a 都成立）

所以 &'static str 可以用在任何需要 &'a str 的地方。
```

---

## 可以，但反过来不行

```rust
fn accept_short<'a>(s: &'a str) {
    println!("{s}");
}

let long_lived: &'static str = "hello"; // 'static 生命周期
accept_short(long_lived);  // ✅ 'static 可以用于任何 'a

fn accept_static(s: &'static str) {
    println!("{s}");
}

let owned = String::from("hi");
// accept_static(&owned); // ❌ &owned 的生命周期不是 'static
```

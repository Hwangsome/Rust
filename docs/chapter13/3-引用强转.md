# 3. 引用强转

> 关键词：&mut → &、Deref、DerefMut

## 一分钟结论

- `&mut T` → `&T`：合法（降级）
- `&mut T` → `&mut U`：要求 `T: DerefMut<Target = U>`
- `&T` → `&U`：要求 `T: Deref<Target = U>`
- **`&T` ↛ `&mut T`**：永远不合法（违反借用规则）

## 重复语法

| 语法 | 含义 |
|-----|-----|
| `$(...)* ` | 零次或多次 |
| `$(...)+ ` | 一次或多次 |
| `$(...),*` | 用 `,` 分隔，零次或多次 |
| `$(...),+` | 用 `,` 分隔，一次或多次 |

---

## 实现 `vec!`

```rust
macro_rules! my_vec {
    ( $( $x:expr ),* $(,)? ) => {{
        let mut v = Vec::new();
        $( v.push($x); )* // 对每个 $x 展开一次
        v
    }};
}

let v1 = my_vec![1, 2, 3];
let v2 = my_vec!["a", "b", "c",]; // 允许尾逗号
```

## 实现 `max!` 宏

```rust
macro_rules! max {
    ($x:expr) => ($x);
    ($x:expr, $($rest:expr),+) => {
        std::cmp::max($x, max!($($rest),+))
    };
}

let result = max!(1, 5, 3, 8, 2);
println!("max = {result}"); // 8
```

## 生成结构体

```rust
macro_rules! define_struct {
    ($name:ident { $( $field:ident : $ty:ty ),* }) => {
        #[derive(Debug)]
        struct $name {
            $( $field: $ty, )*
        }
    };
}

define_struct!(Point { x: f64, y: f64 });
let p = Point { x: 1.0, y: 2.0 };
println!("{p:?}");
```

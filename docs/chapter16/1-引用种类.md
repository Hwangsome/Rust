## Rust 的所有指针/引用类型速查

| 类型 | 大小 | 语义 | 安全 |
|-----|-----|-----|-----|
| `&T` | 8 或 16 字节 | 只读借用 | ✅ |
| `&mut T` | 8 或 16 字节 | 独占可变借用 | ✅ |
| `Box<T>` | 8 或 16 字节 | 堆上独占 | ✅ |
| `Rc<T>` | 8 或 16 字节 | 单线程共享 | ✅（运行时检查）|
| `Arc<T>` | 8 或 16 字节 | 多线程共享 | ✅ |
| `*const T` | 8 或 16 字节 | 裸指针只读 | ⚠️ unsafe |
| `*mut T` | 8 或 16 字节 | 裸指针可写 | ⚠️ unsafe |
| `NonNull<T>` | 8 字节 | 非空裸指针 | ⚠️ unsafe |

## 关键区别

```rust
// &T 是 Copy，赋值后两者都有效
let x = 42;
let r1 = &x;
let r2 = r1;   // 复制引用
println!("{r1} {r2}"); // 都有效

// &mut T 不是 Copy，赋值是 move
let mut y = 42;
let m1 = &mut y;
let m2 = m1;   // move！m1 失效
*m2 += 1;
// println!("{}", *m1); // ❌

// 裸指针不受借用检查
let raw: *const i32 = &x;
let val = unsafe { *raw };  // 需要 unsafe
```

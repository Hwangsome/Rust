## 这一节解决什么问题

Rust 的控制流看起来和 C/Java 差不多，但有几个关键差异：

1. `if` 是**表达式**（不是语句）——可以直接赋值
2. `loop` 可以**返回值**——`break value;`
3. `for` 循环基于**迭代器**——比 C 风格 `for(;;)` 更安全
4. 没有 `do-while`，但 `loop { ... if cond { break }; }` 等价
5. 没有 `switch/case`，但有更强大的 `match`（后续章节）

---

## 一分钟结论

- `if/else if/else` 是表达式，可以 `let x = if cond { a } else { b };`
- 所有分支必须返回**相同类型**（否则编译报错）
- `loop`：无限循环，用 `break` 退出；`break value` 让 loop 产生值
- `while`：条件型循环
- `for x in iter`：迭代器驱动，最推荐的遍历方式
- `for i in 0..5` / `for i in 0..=5`：range 遍历
- `'label: loop`：带标签的循环，`break 'label` 跳出外层

---

## 与其他语言对比

| 特性 | Java/C | Python | Rust |
|-----|-------|--------|------|
| if 是否是表达式 | 否（语句）| 是（三元 `a if c else b`）| **是** |
| for 是否基于迭代器 | 否（可以下标或迭代器）| 是 | **是（总是）**|
| 无限循环 | `while(true)` | `while True:` | `loop {}` |
| break 带返回值 | 否 | 否 | **是**（`break value`）|
| 多层循环 break | 无官方支持（用 flag）| 无官方支持 | **标签** |

## 详细原理

### 1. `if` 作为表达式

```rust
let temperature = 28;

// 赋值
let status = if temperature > 30 {
    "炎热"
} else if temperature > 20 {
    "舒适"
} else {
    "凉爽"
};
println!("天气状态: {status}");

// 也可以直接用在函数调用里
println!("天气状态: {}", if temperature > 30 { "炎热" } else { "其他" });

// ⚠️ 所有分支必须是同一类型
// let bad = if cond { 42 } else { "hello" }; // ❌ i32 vs &str
```

### 2. `loop` + 带值 `break`

```rust
// loop 作为表达式（常见模式：找到满足条件的值）
let mut attempts = 0;
let result = loop {
    attempts += 1;
    if attempts * attempts > 50 {
        break attempts;  // break 带值，loop 表达式的值就是这个
    }
};
println!("第一个平方大于50的自然数: {result}");  // 8（8*8=64>50）

// 通常用于：重试逻辑、游戏主循环、等待条件满足
```

### 3. `while` 循环

```rust
let mut n = 0;
while n < 5 {
    print!("{n} ");
    n += 1;
}
println!();  // 0 1 2 3 4

// while let（模式匹配循环）
let mut stack = vec![1, 2, 3];
while let Some(top) = stack.pop() {
    print!("{top} ");  // 3 2 1（从后往前弹出）
}
println!();
```

### 4. `for` 循环

```rust
// 遍历 Vec（借用）
let fruits = vec!["苹果", "香蕉", "橙子"];
for fruit in &fruits {  // 借用，fruits 仍然有效
    println!("{fruit}");
}

// 遍历 range
for i in 0..5 {    // 0, 1, 2, 3, 4（不包含5）
    print!("{i} ");
}
println!();

for i in 0..=5 {   // 0, 1, 2, 3, 4, 5（包含5）
    print!("{i} ");
}
println!();

// 步进
for i in (0..10).step_by(2) {
    print!("{i} ");  // 0 2 4 6 8
}
println!();

// 反向
for i in (0..5).rev() {
    print!("{i} ");  // 4 3 2 1 0
}
println!();

// 带下标
for (i, fruit) in fruits.iter().enumerate() {
    println!("[{i}] {fruit}");
}
```

### 5. 带标签的循环（多层嵌套）

```rust
'outer: for x in 0..5 {
    for y in 0..5 {
        if x + y > 6 {
            println!("找到: ({x}, {y})");
            break 'outer;  // 跳出外层循环！
        }
    }
}

// continue 也支持标签
'outer: for i in 0..3 {
    for j in 0..3 {
        if j == 1 {
            continue 'outer;  // 跳过内层剩余，进入外层下一次迭代
        }
        println!("({i}, {j})");
    }
}
```

## 完整运行示例

```rust
fn main() {
    println!("=== if 作为表达式 ===");
    let n = 42;
    let parity = if n % 2 == 0 { "偶数" } else { "奇数" };
    let description = format!("{n} 是{}，{}", parity, if n > 0 { "正数" } else { "非正数" });
    println!("{description}");
    println!();

    println!("=== loop + break 带值 ===");
    let mut count = 0;
    let limit = loop {
        count += 1;
        if count * count > 100 {
            break count;  // 第一个平方超过100的数
        }
    };
    println!("第一个平方 > 100 的自然数: {limit} ({}² = {})", limit, limit * limit);
    println!();

    println!("=== while let ===");
    let mut numbers = vec![10, 20, 30, 40, 50];
    let mut sum = 0;
    while let Some(n) = numbers.pop() {
        sum += n;
        print!("{n} ");
    }
    println!();
    println!("总和: {sum}");
    println!();

    println!("=== for range 变体 ===");
    print!("0..5:      "); (0..5).for_each(|i| print!("{i} ")); println!();
    print!("0..=5:     "); (0..=5).for_each(|i| print!("{i} ")); println!();
    print!("step 2:    "); (0..10).step_by(2).for_each(|i| print!("{i} ")); println!();
    print!("reversed:  "); (0..5).rev().for_each(|i| print!("{i} ")); println!();
    println!();

    println!("=== 带标签的嵌套循环 ===");
    let mut found = None;
    'search: for row in 0..5 {
        for col in 0..5 {
            if row * 5 + col == 17 {
                found = Some((row, col));
                break 'search;
            }
        }
    }
    println!("找到 17 的位置: {:?}", found);
    println!();

    println!("=== 迭代集合的常见模式 ===");
    let data = vec![3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5];

    let evens: Vec<_> = data.iter().filter(|&&x| x % 2 == 0).collect();
    println!("偶数: {evens:?}");

    let (positives, zeros_negatives): (Vec<_>, Vec<_>) =
        data.iter().partition(|&&x| x > 0);
    println!("正数: {positives:?}");
}
```

## 注意点与陷阱

### 陷阱 1：if 分支类型不一致

```rust
// ❌ 分支类型不同
let bad = if true { 42 } else { "hello" };
// error: if and else have incompatible types

// ✅ 如果需要不同类型，用 Box<dyn Trait> 或 enum
```

### 陷阱 2：for in collection 默认消耗集合

```rust
let v = vec![1, 2, 3];
for x in v { println!("{x}"); }  // v 被消耗
// println!("{v:?}"); // ❌ v 已被 move

// 避免：用 &v 或 v.iter()
for x in &v { println!("{x}"); }  // 保留 v
```

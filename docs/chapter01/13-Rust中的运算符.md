## 一分钟结论

- Rust 运算符大致可分为：**算术、比较、逻辑、赋值、位运算**
- 大部分和常见语言相似，但不要想当然
- `^` 不是幂运算
- 位运算和逻辑运算是两套完全不同的东西

## 定义

运算符是把值组合成表达式的最基本语法工具。Rust 里最常见的几类：

| 类别 | 示例 |
| --- | --- |
| 算术运算 | `+ - * / %` |
| 比较运算 | `== != < > <= >=` |
| 逻辑运算 | `&& || !` |
| 赋值运算 | `= += -= *= /=` |
| 位运算 | `& | ^ << >>` |

## 作用

运算符的作用不是“让代码更短”，而是：

- 表达数值关系
- 表达真假关系
- 表达状态更新
- 表达二进制位级别的变换

## 原理

### 1. 比较运算返回 `bool`

```rust
let a = 10;
let b = 10;
println!("{}", a == b);
```

这类表达式的结果不是数字，而是 `true` / `false`。

### 2. 逻辑运算作用在布尔表达式上

```rust
(left > 5) && (right < 25)
```

这里的核心不是 `&&` 本身，而是：

- 左边和右边都先是布尔结果
- 再由 `&&` 合并

### 3. 赋值运算是“先算，再写回”

```rust
x += 5;
x *= 2;
```

它们都是普通赋值的简写。

### 4. 位运算直接作用于二进制位

```rust
let bits: u8 = 4;
println!("{}", bits << 1);
```

位运算不是“高级版本的逻辑运算”，而是直接在位层面操作。

## 最小示例

```rust
fn main() {
    println!("17 % 5 = {}", 17 % 5);

    let a = 10;
    let b = 10;
    println!("a == b => {}", a == b);

    let left = 10;
    let right = 20;
    println!("logical && => {}", (left > 5) && (right < 25));

    let mut x = 5;
    x += 5;
    x *= 2;
    println!("assignment result = {}", x);

    let bits: u8 = 4;
    println!("left shift = {}", bits << 1);
}
```

## 注意点

### 1. `^` 不是幂运算

如果你想表达幂，Rust 通常用方法，例如：

```rust
2_i32.pow(3)
```

### 2. 位运算和逻辑运算不要混

- `&&` / `||` 处理布尔语义
- `&` / `|` / `^` 处理位语义

### 3. `%` 是取余，不是“百分比”

它最常用于：

- 整除判断
- 周期性逻辑
- 环状索引

## 常见错误

### ❌ 错误 1：把 `^` 当幂

这是很多其他语言背景读者都会踩的坑。

### ❌ 错误 2：把 `&` 和 `&&` 混为一谈

虽然符号相似，但语义层完全不同。

### ❌ 错误 3：只记符号，不记返回值类型

尤其是比较运算，真正重要的是“它返回 `bool`”。

## 我的理解

- 运算符这节真正要建立的是“分类意识”
- 同样一个符号层，看起来都像“计算”，实际解决的问题完全不同
- 一旦分类清楚，后面再学重载、优先级、结合性就不容易乱

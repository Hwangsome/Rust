# 1. 线程基础

Rust 的线程是 OS 线程。核心 API：`std::thread::spawn` / `JoinHandle::join` / `thread::sleep`。

## 对应代码

- [topic_01_thread_basics.rs](../../chapters/chapter14/src/topic_01_thread_basics.rs)

## 输出要点

- 每个 spawn 返回的 `JoinHandle<T>` 能拿子线程的返回值
- `join()` 阻塞当前线程直到子线程结束
- 不同子线程的打印顺序**不确定**——这是并发程序的常态

## 这一节解决什么问题

Rust 编译器的错误信息比大多数语言都详细，但新手往往不知道如何"阅读"它。学会读错误信息，比记住所有规则更重要。

---

## 如何读 Rust 的错误信息

Rust 错误信息的结构：

```
error[E0382]: borrow of moved value: `s`     ← 错误码 + 简短描述
 --> src/main.rs:5:20                         ← 文件位置
  |
3 | let s1 = String::from("hello");
  |     -- move occurs because `s1` has type `String`
4 | let s2 = s1;
  |          -- value moved here               ← 发生在哪
5 | println!("{}", s1);
  |                ^^ value borrowed here after move  ← 问题在哪
  |
  = help: consider cloning the value if the usage allows
```

五个关键部分：
1. **错误码**（E0xxx）：唯一标识，可以查详细文档
2. **简短描述**：一句话说明问题
3. **位置**：`-->` 后面是文件和行号
4. **代码注释**：箭头 `^` 和 `-` 指向具体位置并解释
5. **`help:`**：编译器的修复建议

## 常见错误码速查

### E0308：类型不匹配

```rust
let x: i32 = "hello"; // ❌ 字符串不能赋给 i32
```

```text
error[E0308]: mismatched types
  expected `i32`, found `&str`
```

### E0382：使用已被 move 的值

```rust
let s = String::from("hi");
let t = s;
println!("{s}"); // ❌ s 已被 move
```

```text
error[E0382]: borrow of moved value: `s`
  value moved here
  ...used here after move
  help: consider cloning the value
```

### E0384：不可变变量重新赋值

```rust
let x = 5;
x = 6; // ❌
```

```text
error[E0384]: cannot assign twice to immutable variable `x`
  help: consider making this binding mutable: `mut x`
```

### E0502：不可变和可变借用冲突

```rust
let mut v = vec![1, 2, 3];
let r = &v;
v.push(4); // ❌ 有不可变借用时不能改
println!("{r:?}");
```

```text
error[E0502]: cannot borrow `v` as mutable because it is also borrowed as immutable
```

### E0499：多个可变借用

```rust
let mut v = vec![1];
let r1 = &mut v;
let r2 = &mut v; // ❌
println!("{r1:?} {r2:?}");
```

```text
error[E0499]: cannot borrow `v` as mutable more than once at a time
```

### E0597：生命周期不够长

```rust
let r;
{
    let x = 5;
    r = &x; // ❌ x 在块结束时消失
}
println!("{r}");
```

```text
error[E0597]: `x` does not live long enough
  `x` dropped here while still borrowed
```


## 完整运行示例

```rust
fn demonstrate_common_errors() {
    println!("=== 模拟常见错误并展示如何修复 ===");

    // 演示 E0382 及其修复
    println!("-- E0382: move 后继续使用 --");
    let s1 = String::from("hello");

    // 错误方式（注释掉，因为会导致编译失败）：
    // let s2 = s1;
    // println!("{s1}"); // ← 报 E0382

    // 修复方式 A：clone
    let s2 = s1.clone();
    println!("clone: s1={s1}, s2={s2}");

    // 修复方式 B：改用借用
    let s3 = &s1;
    println!("borrow: s1={s1}, s3={s3}");

    println!();
    println!("-- E0384: 修改不可变变量 --");
    let mut x = 5;  // 修复：加了 mut
    x += 1;
    println!("x = {x}");

    println!();
    println!("-- E0502: 借用冲突 --");
    let mut v = vec![1, 2, 3];
    {
        let first = &v[0];
        println!("first = {first}");
        // first 借用在这里结束（NLL）
    }
    v.push(4);  // 现在可以 push 了
    println!("v = {v:?}");
}

fn main() {
    demonstrate_common_errors();

    println!();
    println!("=== 遇到编译错误的正确处理步骤 ===");
    println!("1. 读 error[Exxxx]：记下错误码");
    println!("2. 读简短描述：一句话理解问题");
    println!("3. 看 --> 位置：找到问题在哪");
    println!("4. 读代码注释（^ 和 -）：理解为什么");
    println!("5. 读 help:/note:：编译器的建议");
    println!("6. 必要时：rustc --explain Exxxx");
}
```

## 常见错误码与修复方法总结

| 错误码 | 问题 | 快速修复 |
|--------|------|---------|
| E0308 | 类型不匹配 | 检查类型，用 `as` 转换 |
| E0382 | 移动后使用 | 用 `clone()` 或 `&` 借用 |
| E0384 | 修改不可变变量 | 加 `mut` |
| E0425 | 找不到变量/函数 | 检查拼写、作用域、`use` 导入 |
| E0499 | 多个可变借用 | 确保同时只有一个 `&mut` |
| E0502 | 不可变+可变借用冲突 | 调整借用顺序，让不可变借用先结束 |
| E0597 | 生命周期不够长 | 确保引用指向的值活得足够久 |

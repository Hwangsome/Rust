## 这一节解决什么问题

每次遍历集合，你都需要问自己三个问题：

1. 我**需要消耗**这个集合，还是遍历完还要继续用它？
2. 我需要**修改**元素，还是只读？
3. 元素是**值类型**还是**引用类型**？

三种遍历方式恰好对应这三个维度的不同选择。

---

## 一分钟结论

| 方法 | 产出类型 | 是否消耗集合 | 适用场景 |
|-----|---------|------------|--------|
| `.iter()` | `&T` | ❌ | 只读遍历（最常见）|
| `.iter_mut()` | `&mut T` | ❌ | 原地修改元素 |
| `.into_iter()` | `T` | ✅ | 消耗集合，获取元素所有权 |

等价的 `for` 写法：

| 写法 | 等价于 |
|-----|------|
| `for x in &v` | `for x in v.iter()` |
| `for x in &mut v` | `for x in v.iter_mut()` |
| `for x in v` | `for x in v.into_iter()` |

## 详细原理

### 1. `iter()`：最安全，最常用

```rust
let scores = vec![85, 92, 78, 95, 88];

// x 是 &i32
let sum: i32 = scores.iter().sum();
let max = scores.iter().max();

println!("平均分: {:.1}", sum as f64 / scores.len() as f64);
println!("最高分: {:?}", max);
println!("scores 仍然有效: {scores:?}");
```

### 2. `iter_mut()`：原地修改

```rust
let mut prices = vec![100, 200, 300, 400];

// 全部打折 10%
prices.iter_mut().for_each(|p| *p = (*p as f64 * 0.9) as i32);
println!("打折后: {prices:?}");  // [90, 180, 270, 360]

// 等价的 for 写法
for p in &mut prices {
    *p += 10;  // 加上配送费
}
println!("加配送费后: {prices:?}");
```

### 3. `into_iter()`：转移所有权

```rust
let tasks: Vec<String> = vec![
    "买菜".into(), "做饭".into(), "洗碗".into()
];

// 消耗 tasks，把每个 String 发送到线程
let handles: Vec<_> = tasks.into_iter().map(|task| {
    std::thread::spawn(move || {
        println!("执行任务: {task}");
    })
}).collect();

// tasks 已被消耗，不能再用
for h in handles { h.join().ok(); }
```

### 4. HashMap 的遍历

```rust
use std::collections::HashMap;

let mut scores: HashMap<&str, i32> = HashMap::new();
scores.insert("Alice", 95);
scores.insert("Bob", 87);
scores.insert("Carol", 92);

// &HashMap → 产出 (&K, &V)
for (name, score) in &scores {
    println!("{name}: {score}");
}

// &mut HashMap → 产出 (&K, &mut V)（只能改 value，不能改 key）
for (_, score) in &mut scores {
    *score += 5;  // 所有人加5分
}

// into_iter → 产出 (K, V)（消耗 HashMap）
let transformed: HashMap<&str, i32> = scores.into_iter()
    .filter(|(_, v)| *v >= 90)
    .collect();
println!("90分以上: {transformed:?}");
```

## 完整运行示例

```rust
use std::collections::HashMap;

fn main() {
    let data = vec![10, 20, 30, 40, 50];
    let mut mutable_data = data.clone();

    println!("=== iter()：只读，集合保留 ===");
    let squares: Vec<i32> = data.iter().map(|&x| x * x).collect();
    println!("squares: {squares:?}");
    println!("data 完好: {data:?}");
    println!();

    println!("=== iter_mut()：原地修改 ===");
    mutable_data.iter_mut().for_each(|x| *x *= 2);
    println!("doubled: {mutable_data:?}");
    println!();

    println!("=== into_iter()：获取所有权 ===");
    let consumed: Vec<String> = data.into_iter()
        .map(|n| format!("item_{n}"))
        .collect();
    println!("consumed: {consumed:?}");
    // println!("{data:?}"); // ❌ data 已被消耗
    println!();

    println!("=== HashMap 遍历 ===");
    let mut inventory: HashMap<&str, u32> = [
        ("苹果", 50), ("香蕉", 30), ("橙子", 20)
    ].into_iter().collect();

    println!("库存:");
    let mut items: Vec<_> = inventory.iter().collect();
    items.sort_by_key(|(k, _)| *k);
    for (item, qty) in &items {
        println!("  {item}: {qty}件");
    }

    // 卖掉5件苹果
    if let Some(qty) = inventory.get_mut("苹果") {
        *qty -= 5;
    }

    println!("卖出5件苹果后:");
    let total: u32 = inventory.values().sum();
    println!("  总库存: {total}件");
    println!();

    println!("=== 常见模式：enumerate ===");
    let words = ["first", "second", "third", "fourth"];
    for (i, word) in words.iter().enumerate() {
        println!("  [{i}] {word}");
    }

    println!();

    println!("=== 常见模式：zip ===");
    let names = ["Alice", "Bob", "Carol"];
    let scores = [95, 87, 92];
    let combined: Vec<_> = names.iter().zip(scores.iter()).collect();
    for (name, score) in &combined {
        println!("  {name}: {score}");
    }
}
```

## 注意点与陷阱

### 陷阱：`iter_mut()` 不能同时借用多个元素

```rust
let mut v = vec![1, 2, 3, 4, 5];

// ❌ 不能在迭代时同时借用另一个元素
for x in v.iter_mut() {
    // v[0] = 10; // ❌ v 已被 iter_mut 独占借用
    *x += 1;      // ✅ 只操作当前元素
}
```

## 什么是协变

`F<T>` 在 T 上协变（covariant）：如果 `A: B`（A 是 B 的子类型），则 `F<A>: F<B>`。

对生命周期：`F<'long>` 可以用在需要 `F<'short>` 的地方。

---

## 协变的例子

```rust
// &'a T 对 'a 是协变的
fn accept_ref<'a>(_: &'a str) {}

let long_lived: &'static str = "hello";
accept_ref(long_lived); // ✅ 'static 协变到任意 'a

// Vec<T> 对 T 是协变的
fn accept_vec_ref<'a>(_: &Vec<&'a str>) {}
let v: Vec<&'static str> = vec!["hi"];
accept_vec_ref(&v); // ✅
```

---

## 协变适用的类型

`Box<T>`、`Vec<T>`、`Rc<T>`、`Arc<T>`、`&'a T` 等**只读或独占**的容器通常是协变的。

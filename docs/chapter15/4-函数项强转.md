## 函数名是零大小的"函数项类型"

每个具名函数 `fn foo(...)` 都有一个独特的零大小**函数项类型**（function item type）。在需要 `fn(args) -> ret`（函数指针类型）的地方，函数项会自动强转：

```rust
fn double(x: i32) -> i32 { x * 2 }
fn halve(x: i32) -> i32 { x / 2 }

// 数组要求同类型 → 自动强转为 fn 指针
let ops: [fn(i32) -> i32; 2] = [double, halve];
for op in &ops {
    println!("{}", op(10));
}
```

---

## fn 指针 vs 函数项大小

```rust
use std::mem::size_of;
println!("fn 指针大小: {}", size_of::<fn(i32) -> i32>()); // 8
println!("函数项大小: {}", size_of_val(&double));           // 0（函数项是 ZST）
```

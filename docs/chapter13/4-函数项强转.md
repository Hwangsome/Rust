# 4. 函数项（fn item）强转为 fn pointer

> 关键词：fn item、fn pointer、ZST

## 一分钟结论

- 每个具名函数都有一个**独特、零大小的类型**（fn item type）
- 需要 `fn(args) -> ret`（fn pointer）的位置，fn item 自动强转成 fn pointer
- fn pointer 不是零大小（64 位机器上 8 字节）

## 应用

```rust
let ops: [fn(i32, i32) -> i32; 2] = [add, sub];
// 数组要求所有元素同类型，add 和 sub 被统一强转成 fn pointer
```

通过 fn pointer 调用时会走间接分派，比直接调用略慢；但它让我们能把一组函数存在数组/Vec 里。

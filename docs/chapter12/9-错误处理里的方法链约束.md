## 这一节解决什么问题

第 3 章讲了方法链（Builder 模式）。当某些步骤可能失败时，如何保持链式调用的可读性同时处理错误？

关键决策：方法返回什么类型？

| 返回类型 | 能直接链下去 | 场景 |
|---------|-----------|-----|
| `Self` | ✅ | 不会失败的 builder 步骤 |
| `&mut Self` | ✅ | 原地修改，不会失败 |
| `Result<Self>` | 需要 `?` | 可能失败，消费 self |
| `Result<&mut Self>` | 需要 `?` | 可能失败，借用 self |

---

## 详细原理

### 两种带错误的链式风格

```rust
// 风格 A：返回 Result<Self>（消费型，需要函数返回 Result）
#[derive(Debug, Default)]
struct Builder { value: i32 }

impl Builder {
    fn set(mut self, v: i32) -> Result<Self, String> {
        if v < 0 { return Err(format!("值 {v} 不能为负")); }
        self.value = v;
        Ok(self)
    }
    fn build(self) -> i32 { self.value }
}

fn create() -> Result<i32, String> {
    Ok(Builder::default()
        .set(10)?
        .set(20)?
        .build())
}

// 风格 B：返回 &mut Self（借用型，可以不需要函数返回 Result）
#[derive(Debug, Default)]
struct Builder2 { value: i32, errors: Vec<String> }

impl Builder2 {
    fn set(&mut self, v: i32) -> &mut Self {
        if v < 0 { self.errors.push(format!("值 {v} 不能为负")); }
        else { self.value = v; }
        self
    }
    fn build(&self) -> Result<i32, Vec<String>> {
        if self.errors.is_empty() { Ok(self.value) }
        else { Err(self.errors.clone()) }
    }
}
```

## 完整运行示例

```rust
#[derive(Debug, Default)]
struct HttpRequest {
    url: Option<String>,
    method: String,
    headers: std::collections::HashMap<String, String>,
    body: Option<String>,
}

impl HttpRequest {
    fn new() -> Self {
        HttpRequest { method: "GET".to_string(), ..Default::default() }
    }

    fn url(mut self, url: &str) -> Result<Self, String> {
        if !url.starts_with("http") {
            return Err(format!("无效 URL，必须以 http 开头: {url}"));
        }
        self.url = Some(url.to_string());
        Ok(self)
    }

    fn method(mut self, method: &str) -> Result<Self, String> {
        let valid = ["GET", "POST", "PUT", "DELETE", "PATCH"];
        if !valid.contains(&method) {
            return Err(format!("不支持的 HTTP 方法: {method}"));
        }
        self.method = method.to_string();
        Ok(self)
    }

    fn header(mut self, key: &str, value: &str) -> Self {
        // 头部添加不会失败
        self.headers.insert(key.to_string(), value.to_string());
        self
    }

    fn body(mut self, body: &str) -> Self {
        self.body = Some(body.to_string());
        self
    }

    fn send(&self) -> Result<String, String> {
        let url = self.url.as_ref().ok_or("缺少 URL")?;
        Ok(format!("{} {} → (模拟响应 200 OK)", self.method, url))
    }
}

fn build_and_send(url: &str, method: &str) -> Result<String, String> {
    HttpRequest::new()
        .url(url)?          // 可能失败
        .method(method)?    // 可能失败
        .header("Accept", "application/json")  // 不会失败
        .header("User-Agent", "RustBot/1.0")
        .send()
}

fn main() {
    let cases = [
        ("https://api.example.com/users", "GET"),
        ("ftp://invalid.com", "GET"),    // 无效 URL
        ("https://api.example.com/data", "TRACE"), // 不支持的方法
        ("https://api.example.com/items", "POST"),
    ];

    for (url, method) in cases {
        match build_and_send(url, method) {
            Ok(response) => println!("✅ {response}"),
            Err(e) => println!("❌ {e}"),
        }
    }
}
```

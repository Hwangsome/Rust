## 带泛型参数的方法不能用于 dyn Trait

```rust
trait NotObjectSafe {
    fn generic_method<T>(&self, x: T); // ❌ 泛型参数，需要单态化
}

// ❌ 报错 E0038
// let v: Box<dyn NotObjectSafe> = ...;

// ✅ 把泛型参数移到 trait 级别（关联类型）
trait ObjectSafe {
    type T;
    fn method(&self, x: &Self::T);
}
```

---

## 或者用 where Self: Sized 排除

```rust
trait Hybrid {
    fn normal(&self) -> i32;       // ✅ 对象安全

    fn generic<T: Clone>(&self, x: T) where Self: Sized; // 被排除出 dyn
}

// 现在 Hybrid 是对象安全的（dyn Hybrid 只能调 normal）
fn use_dyn(obj: &dyn Hybrid) {
    let _ = obj.normal();
    // obj.generic(42); // ❌ dyn 没有这个方法
}
```

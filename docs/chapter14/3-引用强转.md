## `&mut T` 可以被强转为 `&T`，反之不行

这是合理的：降级（从更强的到更弱的）是安全的；升级则可能违反借用规则。

```rust
fn read_only(v: &Vec<i32>) { println!("{v:?}"); }

let mut v = vec![1, 2, 3];
read_only(&mut v); // ✅ &mut Vec<i32> → &Vec<i32>（降级，安全）
read_only(&v);     // ✅ 直接传 & 也可以

// 反向不行：
fn requires_mut(v: &mut Vec<i32>) { v.push(0); }
// requires_mut(&v); // ❌ &Vec<i32> 不能升级为 &mut Vec<i32>
```

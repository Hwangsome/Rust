# 13. 项目：并发"抓取"

用 `thread::spawn` + `mpsc::channel` 并行跑多个"抓取"任务。本 demo 没有发真实网络，用 `sleep` 模拟。

## 对应代码

- [topic_13_project_webscrapping_using_threads.rs](../../chapters/chapter14/src/topic_13_project_webscrapping_using_threads.rs)

## 设计点

- 每个子线程返回 `Result<...>`，失败只影响自己
- 主线程用 `for (url, res) in rx` 收集
- 原 `tx` 显式 `drop(tx)`，让 rx 在所有子线程结束后自然退出
